# Guide - fast start

## :window: Windows
```
git clone https://github.com/TouhruCreative/DjangoProjectDeliveryShop.git
cd FullStuckTestProject
python -m venv venv
venv\Scripts\activate
pip install -r requirements.txt
```

## :apple: MacOS / :penguin: Linux
```
git clone https://github.com/TouhruCreative/DjangoProjectDeliveryShop.git
cd FullStuckTestProject
python -m venv venv
source venv/bin/activate
pip install -r requirements.txt
```
